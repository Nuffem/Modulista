# Modulista

Modulista é um web SPA para análise de dados.

A página inicial é do tipo **Lista**.

## Definições

### Lista
Página que possui itens.
Pode incluir itens do tipo:
- Lógico
- Número
- Texto
- Lista
- Função
- Aplicação

### Lógico
Item que contém um campo de valor lógico.

### Número
Item que contém um campo de número.

### Texto
Item que contém um campo de texto.

### Função
Página que possui parâmetros (Lista) e retorno (Valor).

### Aplicação
Página que possui referência de uma função e argumentos (Lista).

### Valor
Item que contém um campo para selecionar o tipo e um campo para informar o valor (que modifica de acordo com o tipo selecionado).
